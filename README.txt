1) Login into cs1.utdallas.edu with Net-Id and Password.
2) Login into 12 different distributed computing machines with Net-Id and Password.
3) To compile the project:
   javac *.java
4) To run all the 7 server machines- run the following command: java Caller
   Enter s1 to s7 for each server respectively.
5) To run each of the 5 clients- run the following command: java Caller
   Enter c1 to c5 for each client respectively.
